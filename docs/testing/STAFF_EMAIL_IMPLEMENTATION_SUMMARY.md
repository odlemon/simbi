```
Seller Creates Staff
        ↓
System Validates Input
        ↓
Generate Secure 12-Char Password
        ↓
Hash Password (bcrypt)
        ↓
Create Staff Record in DB
        ↓
Log Activity (STAFF_CREATED)
        ↓
Send Credentials Email via ZeptoMail
        ↓
Log Email Status
        ↓
Return Response (staff + tempPassword)
        ↓
Staff Receives Email
        ↓
Staff Logs In with Credentials
        ↓
✅ DONE!
```

## 🔐 **Password Security**

### **Generation:**
- **Length:** 12 characters
- **Character Sets:** 4 (Upper, Lower, Numbers, Special)
- **Method:** `crypto.randomInt()` (cryptographically secure)
- **Ambiguous Characters Excluded:** I, O, l, 0, 1

### **Storage:**
- **Hashing:** bcrypt with 10 rounds
- **Not Stored Plain:** Only hash in database
- **Temp Password:** Only in API response, not persisted
